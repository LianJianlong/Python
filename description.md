# 模块大纲
## 项目信息模块
1、一级文件夹-项目名称
2、二级文件夹-部门名称
3、三级表格-名称自定义
4、不同的表之间数据可以实现互相引用，类似数据库的外键
5、表头可以选择筛选，排序，聚合，归类等功能
6、表头可以变化视图，视图分为日历视图，gantt视图，照片墙等（新视图有待增加）
## 项目监控模块
1、综合看板
2、自定义，图示类型，数据选择
## 项目工具模块
1、项目工具由高级权限的用户编写，普通用户只能输入信息，不得修改表格模版，格式，计算方法，数据类型等
